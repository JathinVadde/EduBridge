# EduBridge: US High School Guide for International Students

EduBridge is an app with 15 features made to help international and new students understand how US high schools work. 

##  How I Built the Code (Java to Dart)
I first wrote the logic for the calculators in **Java** because that is what I learned in my **AP Computer Science A** class. Then, I self-studied **Dart** and found similarities between the two languages. I translated my Java logic into Dart to make the features work inside **FlutterFlow**.

##  Code Files in This Repository
Since I used the free version of FlutterFlow, I uploaded my custom-coded logic files here to show my work:
* `gpa_calculator.dart`: Calculates weighted and unweighted GPAs.
* `credit_checker.dart`: Checks if a student has enough credits to graduate high school.

##  Complete 15-Feature List
My app has 15 different features to guide new students:
1. **Interactive GPA Calculator** (Custom Coded)
2. **Graduation Credits Tracker** (Custom Coded)
3. **Attendance Rules in U.S. Schools**
4. **Standardized Test Guide** (SAT, ACT, and AP Exams)
5. **US College Applications Overview**
6. **GPA Explained**
7. **Credits & Graduation Requirements**
8. **AP vs Honors vs Regular Classes**
9. **Academic Integrity** (Cheating & Plagiarism)
10. **School Culture in the U.S.**
11. **Extracurricular Activities**
12. **ESL Support Basics**
13. **Report Cards & Transcripts**
14. **Making Friends in U.S. Schools**
15. **Time Management Tips**
