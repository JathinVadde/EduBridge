# EduBridge: US High School Orientation & Guidance Platform

EduBridge is a comprehensive 15-feature guidance mobile application designed to help incoming international and new students seamlessly transition into the United States secondary education system. 

## 🧠 Engineering & Algorithmic Process (Java to Dart)
As part of my development pipeline, I engineered the core math engines by first prototyping the algorithms in **Java** to align with my foundational training in **AP Computer Science A**. After validating the structural logic, I self-studied **Dart** syntax to translate, optimize, and refactor the backend calculation modules for deployment within a production-ready **FlutterFlow** mobile framework.

## 📂 Core Code Modules in This Repository
Because this project is built on a visual development tier, this repository serves as a dedicated source directory highlighting my custom-written algorithms:
* `gpa_calculator.dart`: Manages complex weighted and unweighted high school GPA metrics based on variable course distributions.
* `credit_checker.dart`: Processes user coursework histories against regional high school graduation mandates to output real-time credit tracking diagnostics.
